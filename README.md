# MCP Servers Directory

This repository is a collection of Model Context Protocol (MCP) servers.

## Available Servers

- **[OneNote MCP](./onenote-mcp)**: An MCP server that connects Claude with Microsoft OneNote via the Microsoft Graph API. It enables searching, reading, creating, and updating OneNote pages.

## Adding a New Server

To add a new MCP server:
1. Create a subdirectory under this root folder.
2. Initialize your project inside that folder (e.g., using `npm init` or `npm init @modelcontextprotocol/server`).
3. Add a link to your server in this `README.md` file.
